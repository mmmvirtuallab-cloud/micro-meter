Apparatus Required

  • Outside micrometer (0–25 mm range)
  • Clean cloth or tissue paper
  • Standard gauge block
  • Small brush
  • Oil

Step-by-Step Procedure

  1. Clean the anvil, spindle, and workpiece properly.
  2. Check the zero error by closing the micrometer gently.
  3. Place the object between the anvil and spindle.
  4. Rotate the thimble till the spindle just touches the object.
  5. Use the ratchet stop for uniform pressure.
  6. Lock the spindle using the lock nut & Note the main scale and thimble readings
  7. Calculate total reading and correct for zero error if any.
